Quick Start

快速上手

三步开始使用 GuYu Mate

1

下载安装 GuYu Mate

根据你的操作系统下载对应的安装包，安装后打开软件。首次启动会自动进入一键安装向导。

2

一键安装 Claude Code CLI

在「一键安装」页面，软件会自动检测 Node.js 环境。如果未安装，点击安装按钮即可。然后一键安装 Claude Code CLI 工具。

3

配置谷雨大模型 API Key

前往 谷雨大模型平台 注册账号并生成 API Key，粘贴到软件中即可完成配置。打开终端输入 claude 开始对话！

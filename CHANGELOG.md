# Changelog

## v1.0.0 (2026-03-01)

GuYu Mate 首个正式版本。基于 CC Switch 二次开发，专注 Claude Code CLI 配置管理。

### 功能

- 供应商管理：一键切换 Claude Code API 配置
- MCP 服务器统一管理
- Prompts 多预设管理（Markdown 编辑器）
- Skills 仓库扫描与一键安装
- 深链接协议 `guyumate://` 一键导入
- WebDAV 跨设备同步
- 一键 Node.js 环境安装
- 系统托盘快速切换
- 内置自动更新
- 预设供应商：谷雨大模型
- 支持 Windows / macOS / Linux
- 中文 / 英文 / 日文界面
